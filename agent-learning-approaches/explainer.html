The Results: Hybrids Lead

Scores averaged across all 14 tasks, including coding-agent tasks where CLL scores only 0.200. On 8 core tasks alone, CLL leads with 0.953. But the full picture matters: CLL+RAG (0.926) and Debate (0.925) are the most robust across all task types. ICL (0.710) is never sufficient alone.

Drift Detection: The Key Moment

At step 100, the environment shifts. Without drift detection, CLL clings to old rules and drops to 0.850. With drift detection, it fires, clears stale rules, relearns in 8 steps, and recovers to 1.000. Zero cost on stable tasks.

The Ablation: Three Orthogonal Improvements

Three CLL v2 improvements. Each addresses a different failure mode. Toggle them off to see the impact. None degrades performance on tasks that don't need them. Orthogonal by design.

Debate: The Robust Allrounder

Debate combines CLL and RAG agents with a trust-weighted consensus model. It never drops below rank #4 on any task, placing #2-3 on most. Two agents propose, a trust model arbitrates. Robust by architecture -- no single point of failure.

Not Just Better -- Provably Better

McNemar's chi-squared test confirms: CLL vs RAG on sequence learning (chi2 = 36.98, p < 0.0001). Bootstrap 95% CIs never overlap on key tasks. This is not noise -- it is a statistically significant structural advantage.
