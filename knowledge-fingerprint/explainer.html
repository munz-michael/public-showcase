Agent Identity

The question nobody else is asking

Every agent evaluation tool measures how good the output is. None asks what kind of knowledge does this agent produce?

KFP applies the same 7-dimensional algebra to agent outputs. An agent's identity is its portfolio — the distribution across all its outputs.
