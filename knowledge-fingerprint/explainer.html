Agent Identity

The question nobody else is asking

Every agent evaluation tool measures how good the output is. None asks what kind of knowledge does this agent produce?

KFP applies the same 7-dimensional algebra to agent outputs. An agent's identity is its portfolio: the distribution of knowledge types across all its outputs.

Why this matters

A "Strategist" producing 67% speculative output and zero predictive claims is not doing strategy — it's guessing. The consensus mechanism sees "four agents agreed" but cannot see that three produce identical knowledge types with five blind spots.

Related Work

Building on established foundations
GRADE Framework — Clinical evidence grading. Inspiration for the Evidence dimension. Single dimension; KFP extends to seven.
PROV-O (W3C) — Provenance tracking. Complementary: describes how, not what kind.
Dublin Core / Schema.org — Metadata for containers, not knowledge content.
Semantic Hashing (Salakhutdinov & Hinton) — ML-based binary codes. Opaque, no algebra.
IBM Evidence Grading (US2019251452A1) — Single dimension, no composition.
Epistemic Logic (Hintikka) — Theoretical foundation, not implementation-oriented.
What KFP adds: No prior system combines multi-dimensional epistemological typing + closed composition algebra + cryptographic verification.

Limitations & Collaborators
